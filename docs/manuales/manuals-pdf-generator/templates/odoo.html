Acceso a la aplicación
Inicio de sesión y acceso al menú principal
Pasos para iniciar sesión
1
Abre el navegador y navega a http://localhost:8069
2
Introduce el usuario (email o admin) y la contraseña

Renaix Odoo Web Gestión de productos
Gestión de productos
Crear, editar y gestionar el ciclo de vida de productos
Ciclo de vida del producto
Borrador→ Disponible→ Reservado→ Vendido
Crear nuevo producto
1
Ve a Productos y haz clic en el botón "Nuevo"
2
Rellena los campos obligatorios: Nombre, Precio, Estado del Producto, Propietario
3
Opcionalmente: Descripción, Antigüedad, Ubicación, Categoría, Etiquetas
4
Haz clic en "Guardar" (icono nube) o pulsa Ctrl+S

Solo los administradores pueden eliminar permanentemente. Como alternativa, cambia el estado a Eliminado para ocultarlo sin borrarlo.

Renaix Odoo Web Subida y gestión de imágenes
Subida y gestión de imágenes
Asociar y gestionar imágenes de los productos
Añadir imágenes a un producto
1
Abre el producto y localiza la pestaña o sección "Imágenes"
2
Haz clic en "Agregar una línea" o en el icono de cámara
3
Selecciona un archivo desde tu equipo (JPG / PNG, máximo 5 MB)
4
Activa "Es Principal" en la imagen que quieres como portada del producto
5
Haz clic en "Guardar" para confirmar los cambios

Para gestionar todas las imágenes del sistema en un solo lugar, ve a Productos → Imágenes de Productos. Desde ahí puedes filtrar por producto y eliminar imágenes obsoletas.

Renaix Odoo Web Gestión de etiquetas
Gestión de etiquetas
Crear y asignar etiquetas para clasificar productos
Crear una nueva etiqueta
1
Ve a Configuración → Etiquetas y haz clic en "Nuevo"
2
Introduce el Nombre — se normaliza automáticamente a minúsculas sin espacios
3
Selecciona un Color identificativo (opcional) y haz clic en "Guardar"
Asignar etiquetas a un producto
1
Abre el producto desde Productos
2
En el campo "Etiquetas", escribe o selecciona las etiquetas del desplegable
3
Guarda el producto (Ctrl+S)

Reglas de las etiquetas: mínimo 2 caracteres · máximo 30 · sin duplicados (insensible a mayúsculas) · máximo 5 etiquetas por producto.

Renaix Odoo Web Denuncias e incidencias
Denuncias e incidencias
Revisar y gestionar las denuncias enviadas desde la app
Estados de una denuncia
Pendiente→ En Revisión→ Resuelta / Rechazada
Gestionar una denuncia
1
Accede a Denuncias y abre la denuncia pendiente
2
Haz clic en "Asignar a mí" — el estado cambia a En Revisión
3
Revisa el elemento denunciado desde el enlace en el formulario y rellena el campo "Resolución"
4
Haz clic en "Resolver" o "Rechazar" según proceda

Renaix Odoo Web Errores habituales
Errores habituales y soluciones
Resolución de los problemas más frecuentes
?
La página no carga / Error de conexión
1
Ejecuta docker ps y verifica que el contenedor odoo aparece como Up
2
Si no está activo: docker-compose up -d en la carpeta erp/docker/
3
Espera 30–60 segundos y recarga la página
?
"Acceso denegado" al abrir un menú
Tu usuario no tiene el rol adecuado. Roles disponibles:
Usuario — solo lectura Moderador — gestión de denuncias Administrador — acceso total
?
La imagen no se sube correctamente
Verifica que el archivo es JPG o PNG y no supera 5 MB. Recarga la página e inténtalo de nuevo. Si persiste, comprueba que el servidor Odoo tiene espacio en disco.
?
El formulario no guarda / Error de validación
Lee el mensaje de error en rojo. Casos comunes: "Ya existe una etiqueta con este nombre" · "El motivo debe tener al menos 10 caracteres" · campos obligatorios vacíos.

Renaix Odoo Web Glosario
Glosario
Términos y conceptos clave del sistema
Producto
Artículo de segunda mano publicado por un usuario en Renaix para su venta.
Etiqueta
Palabra clave normalizada (#gaming, #vintage) que clasifica un producto. Máximo 5 por producto.
Estado del producto
Condición física: Nuevo, Como Nuevo, Buen Estado, Estado Aceptable, Para Reparar.
Estado de venta
Ciclo de vida: Borrador, Disponible, Reservado, Vendido, Eliminado.
Denuncia
Reporte de un usuario sobre producto, comentario u otro usuario que infringe las normas.
Moderador
Empleado con acceso para gestionar denuncias, editar productos y revisar comentarios.
Administrador
Empleado con acceso completo a todos los módulos y configuraciones del sistema.
Chatter
Panel de historial de cambios y mensajes internos en la parte inferior de cada formulario Odoo.
Valoración
Puntuación 1–5 estrellas que un comprador otorga a un vendedor tras una transacción.
Categoría
Clasificación principal del producto (Electrónica, Ropa, Deporte…). Gestionada desde Configuración.
JWT
JSON Web Token. Sistema de autenticación segura usado por la API REST de la app móvil.
